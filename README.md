# SLGM

Code and datasets of our paper "On the Strength of Sequence Labeling and Generative Models for Aspect Sentiment Triplet Extraction" accepted by Findings of ACL 2023.

### 1. Requirements

- transformers==4.15.0
- torch==1.7.1

### 2.Usage

```bash
cd data
unzip ASTE.zip
bash scripts/run_single.bash
```
